# Valorant Agent Randomizer

An advanced VALORANT-themed web experience that dynamically randomizes agents with cinematic animations, immersive sound design, voice lines, and an integrated music player.

Live Website:
[https://vzefoq.github.io/Valorant-Agent-Randomizer/](https://vzefoq.github.io/Valorant-Agent-Randomizer/)

---

## Purpose

This project is designed to quickly generate a random VALORANT agent for:

* Fun custom games
* Practice sessions
* Challenge runs
* Scrims
* Team compositions
* Role-based training

Whether you're playing casually or competitively, it adds an immersive twist to agent selection.

---

## Core Features

### Tactical UI Experience

* Full VALORANT-inspired animated interface
* Cinematic agent reveal sequence
* Glitch and shimmer text effects
* Dynamic stat animations

### Role Filtering

Filter agents by role:

* ALL
* DUELISTS
* INITIATORS
* CONTROLLERS
* SENTINELS

### Custom Agent Pool System

Full control over which agents can be randomized:

* Select specific agents
* Exclude specific agents
* SELECT ALL
* DESELECT ALL
* SELECT CURRENT FILTER

Ideal for role-only challenges or limited agent pools.

### Agent Reveal Details

Each randomization displays:

* Agent portrait
* Role badge and origin
* Animated stat breakdown:

  * Aggression
  * Utility
  * Mobility
* Ability cards (C, Q, E, X)
* Agent quote
* Agent voice line on reveal

### Audio System

* Custom spin and reveal sound effects
* Agent voice lines
* Interactive click visual effects on the randomize button

### Integrated Music Player

Built-in music system including:

* Track information display
* Play / Pause
* Next / Previous
* Seekable progress bar
* Volume control and mute
* Collapsible player interface

### Responsive Design

Optimized layout for:

* Desktop
* Tablet
* Mobile devices
