# Valorant Agent Randomizer

A custom VALORANT-themed web app that randomizes agents with animated visuals, sound effects, voice lines, and a built-in music player.

Live page: https://vzefoq.github.io/Valorant-Agent-Randomizer/

## What This Is For

This project helps you quickly get a random VALORANT agent for fun games, practice sessions, challenge runs, or team comps.

## Features

- Full VALORANT-style UI with animated agent reveal
- Role filters:
- `ALL`
- `DUELISTS`
- `INITIATORS`
- `CONTROLLERS`
- `SENTINELS`
- Custom agent pool selector:
- Pick specific agents to include/exclude
- `SELECT ALL`, `DESELECT ALL`, and `CURRENT FILTER` actions
- Agent reveal details:
- Agent portrait
- Role badge and origin
- Aggression, Utility, Mobility stats
- Ability cards (`C`, `Q`, `E`, `X`)
- Agent quote
- Agent voice lines on reveal
- Custom sound system for spin/reveal effects
- Click-position VFX on randomize button
- Built-in music player with:
- Track info
- Play / pause / next / previous
- Progress bar and seek
- Volume + mute
- Collapsible player
- Responsive layout for desktop and mobile
